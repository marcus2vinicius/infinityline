# InfinityLine web scraping

Este projeto tem como finalidade acessar o site da infinity line www.officeinfinity.org e exportar todos os contatos dos usuarios de sua rede em um formato que poderá ser importado no seu celular, alem de te ajudar a entender os problemas da sua rede, mostrando todos os seus downlines e informando quanto cada um precisa indicar.
  - Este projeto esta hospedado em https://infinitylineweb.herokuapp.com
  - Acesse o link para entender a ferramenta http://grupoinfinitydicas.blogspot.com.br
